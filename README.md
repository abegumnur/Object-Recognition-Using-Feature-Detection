# Object Recognition Using Feature Detection

## Overview

This project involves object recognition using four feature detectors: SIFT, ORB, AKAZE, and BRISK, aiming to classify 10 objects from the RGBD dataset.

## Selected Objects

- Lime 1
- Bowl 4
- Lemon 1
- Instant Noodles 1
- Ball 1
- Orange 2
- Toothpaste 2
- Peach 3
- Water Bottle 1
- Scissors 3
